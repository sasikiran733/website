# J Sasi Kiran - Portfolio Website

A professional, responsive portfolio website built with React, Tailwind CSS, and Framer Motion. This project showcases education, experience, skills, and contact information with a modern, glassmorphic design.

## Features

- **Responsive Design:** Fully fluid layout that looks great on mobile, tablet, and desktop.
- **Modern UI:** Utilizes glassmorphism, gradients, and subtle background patterns.
- **Animations:** Smooth entry animations and interactions using Framer Motion.
- **Components:**
  - **Hero:** Dynamic introduction with typing text effect and profile image.
  - **About:** Personal details and objective formatted in clean cards.
  - **Education:** Vertical timeline showcasing academic history.
  - **Experience:** Detailed cards for professional experience.
  - **Contact:** Interactive-looking form and contact details.

## Technologies Used

- **React:** UI Library.
- **Tailwind CSS:** Utility-first styling.
- **Framer Motion:** Animation library.
- **Lucide React:** Iconography.
- **Vite:** Build tool (assumed environment).

## Setup & Running

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Add Profile Photo:**
    - Place your image named `photo.jpeg` in the root directory (same level as `index.html`).

3.  **Run Development Server:**
    ```bash
    npm run dev
    ```

4.  **Build for Production:**
    ```bash
    npm run build
    ```

## Customization

- **Content:** Update `types.ts` or individual components to change text data.
- **Styles:** Modify `tailwind.config` in `index.html` to change the color scheme.
- **Images:** Replace `photo.jpeg` with your own portrait.
